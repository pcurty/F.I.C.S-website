## 🛰️ Présentation du Projet
Ce dépôt contient le code source de la plateforme web de **F.I.C.S. Ingénierie**. L'objectif du site est de présenter nos solutions de "Sur-Mesure Standardisé" pour les professionnels du divertissement immersif.

L'esthétique du site est directement inspirée de l'univers industriel et brutaliste de la mégacorporation **FICSIT Inc.**, mettant en avant la fiabilité, la robustesse et l'optimisation.

## 🛠️ Stack Technique
Pour garantir une performance nominale et une interface fluide, le site utilise les technologies suivantes :
* **HTML5 / CSS3** : Structure sémantique et design industriel.
* **JavaScript (Vanilla)** : Gestion de l'horloge temps réel, des animations au scroll et du moteur de diagnostic.
* **Google Fonts** : Utilisation de *Syncopate* (Titres) et *JetBrains Mono* (Données techniques).
* **Intersection Observer API** : Révélation progressive des modules au défilement.

## ⚙️ Architecture du Système
Le site est conçu comme un "Terminal de Diagnostic" :
* **Header** : Navigation rapide vers les briques technologiques.
* **Hero Section** : Accroche B2B pour la conversion immédiate.
* **Diagnostic Bar** : Barre de statut système en temps réel (Uptime, Status, Clock).
* **Formulaire d'Audit** : Tunnel de capture de prospects pour devis d'ingénierie.
